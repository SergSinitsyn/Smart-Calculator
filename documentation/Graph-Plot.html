Построение графиков

  • Поддерижвается построение графика функции, заданной с помощью выражения в инфиксной нотации с переменной x.
  • Для построения графика функции необходимо дополнительно указать отображаемые область определения и область значения.
  • Для построения графика функции необходимо нажать кнопку Print Graph.
  • Для удаления графика выбрать функцию графика и нажать Delete.
  • Для изменения цвета графика выбрать функцию графика и нажать Color.
  • Максимальное число графиков: 8.
